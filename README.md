📋 How to Start
As soon as you run the console (Ctrl+F5), the game menu will open 🖥️
At the main menu, you’ll have 2 options:

1️⃣ Open the Guide – Learn how to play the game 📖
2️⃣ Start the Game – Jump right into the action! 🎮

That’s it!
